# with_recursive_tree

* Es una extensión de ActiveRecord que usa WITH RECURSIVE para obtener todo el árbol con una sola consulta
* Rápida lectura: 1 QUERY para todo el árbol
* Rápida escritura: 1 QUERY para insertar un nodo
* Misma interface que el resto de las alternativas
* Soporta
  * Postgres (>= 13)
  * MySQL (>= 8.0)
  * SQLite 3 (>= 3.34)
