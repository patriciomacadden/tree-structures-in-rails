# Sinaptia Branding

All slides automatically use Sinaptia brand guidelines:

### Colors
- **Blue (#55adff)** - Primary accent, links, active states
- **Green (#55ffcc / #7fe2c5)** - Links and code strings
- **Grayscale** - Professional, clean backgrounds and text

### Typography
- **PP Monument Extended** - Titles (H1, H2)
- **Inconsolata** - Code and labels (H3, H4)
- **Inter** - Body text and paragraphs

### Design Principles
Technical excellence, innovation, clarity, professionalism
