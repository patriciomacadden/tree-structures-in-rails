# Multi-Section Example

## Section 1: Planning
Start by organizing your content into logical sections. Each section can have its own H2 heading.

## Section 2: Writing
Write your markdown using the full range of supported elements: headings, lists, code blocks, and more.

## Section 3: Building
Run `rake build` or `rake server` to generate your presentation.

## Section 4: Presenting
Navigate with arrow keys and present your beautifully branded slides.
